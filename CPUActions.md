# Pep9CPU Actions
An inventory of all the actions that a user can perform in the Pep9CPU Desktop App.  
Note that all items in italics are also featured on the toolbar.

#### File Menu
- *New*
- *Open*
- *Save*
- Print Microcode  
- Print Object Code  

#### Edit Menu
- *Undo*
- *Redo*
- Cut
- Copy
- Paste
- Un/Comment Code
- Auto-Format Microcode
- Remove Error Messages

#### System Menu
- *Run*
- *Start Debugging*
- *Stop Debugging*
- Clear CPU
- Clear Memory
- One-byte Data Bus
- Switch to Two-byte Data Bus

#### Help Menu
- Using Pep/9 CPU
- Interactive Use
- Microcode Use
- Debugging Use
- Pep/9 Reference
- Examples
